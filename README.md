## 关于『古国简展』
**[古国简展](https://historyapp.qqzhi.cc/)是一个开放历史类数据项目  
在这里，我们收录各个古代国家的时空事件、地理信息  
并在地图图像上将其展现出来  
欢迎共同完善这个项目！**
